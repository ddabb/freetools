# 公众号配图增强版 PRD

## 1. 概述与目标

### 产品定位
公众号配图工具，帮助用户快速生成符合微信公众平台标准的封面图和配图，适用于文章封面、朋友圈分享、公众号次图等多种场景。

### 现状问题
当前版本仅有纯色渐变背景 + 文字，功能过于单一，与同类工具相比缺乏竞争力，用户使用场景受限。

### 优化目标
在保持轻量的前提下，丰富视觉表达，支持背景图、装饰元素、多种比例，接近"轻量版 Canva"的体验。

---

## 2. 功能需求

### 2.1 图片尺寸支持

| 类型 | 比例 | 尺寸（px）| 用途 |
|------|------|----------|------|
| 封面大图 | 2.35:1 | 750×320 | 公众号文章封面 |
| 次条封面 | 1:1 | 500×500 | 文章列表小图 |
| 朋友圈 | 3:4 | 540×720 | 朋友圈分享图 |
| 公众号横幅 | 5:2 | 900×360 | 顶部Banner |

### 2.2 背景系统

#### 2.2.1 纯色/渐变背景（现有）
保留现有 8 种渐变色方案，增加更多配色方案至 16 种。

#### 2.2.2 内置背景图库
提供分类背景图，用户可直接选用，无需上传。

**分类：**
- 简约抽象（几何图形、渐变艺术）
- 自然风光（天空、山水、植物）
- 商务职场（办公室、建筑、图表）
- 节日氛围（春节、中秋、国庆、圣诞）
- 情感励志（日出、星空、跑步）
- 美食生活（咖啡、烘焙、旅行）
- 科技数码（代码、网络、数据）
- 教育学习（书本、课堂、毕业）

背景图存储于小程序静态资源，CDN 加速加载。

#### 2.2.3 自定义上传背景
用户可上传自己的图片作为背景，支持裁剪适配。

### 2.3 装饰元素

#### 2.3.1 Emoji 装饰
在画布上添加 emoji 作为装饰元素：
- 预设 emoji 库（按分类：节日、心情、符号、箭头）
- 支持添加多个 emoji
- 拖拽定位
- 调整大小和透明度

#### 2.3.2 图形装饰
- 几何形状（圆形、方形、三角形、线条）
- 分隔线样式
- 边框装饰

#### 2.3.3 光效/阴影
- 文字阴影
- 元素光晕效果

### 2.4 预设模板

提供若干高质感模板，一键套用：

| 模板名称 | 适用场景 | 特点 |
|---------|---------|------|
| 简约日报 | 资讯/日报 | 留白多，标题突出 |
| 节日庆典 | 节日推文 | 色彩丰富，氛围感强 |
| 商务报告 | 职场/商务 | 专业稳重 |
| 情感生活 | 个人随笔 | 温暖柔和 |
| 科技前沿 | 数码/互联网 | 冷色调，潮流感 |
| 知识干货 | 教程/科普 | 结构清晰 |
| 产品推广 | 营销/种草 | 促销感强 |
| 深夜电台 | 情感/夜读 | 深色背景，文艺感 |

每个模板定义：背景样式 + 文字排版 + 装饰元素 + 字体颜色。

### 2.5 文字排版增强

- 副标题/正文多行文本支持
- 字数限制提示和自动换行
- 文字对齐方式（居左/居中/居右）
- 文字加粗/斜体支持
- 描边/阴影效果

### 2.6 输出与分享

| 功能 | 说明 |
|------|------|
| 保存到相册 | 现有功能 |
| 复制到剪贴板 | 快速分享到其他应用 |
| 一键生成朋友圈文案 | 配合封面自动生成配套文案 |
| 多尺寸导出 | 一次设计，导出多个尺寸 |

---

## 3. 非功能需求

### 3.1 性能
- 背景图懒加载，首屏加载 < 1s
- Canvas 绘制 FPS ≥ 30
- 图片压缩至 200KB 以下，保证分享流畅

### 3.2 兼容性
- 支持微信小程序 canvas 2D API
- 适配不同手机分辨率（iPhone SE 到旗舰机）

### 3.3 维护性
- 背景图资源与代码分离，便于更新
- 模板配置可扩展，不硬编码

## 4. 页面结构

```
公众号配图
├── 顶部 Tab 切换：封面 / 次图 / 朋友圈 / 横幅
├── 预览区域（实时 Canvas 渲染）
├── 快捷模板选择（横向滚动）
├── 编辑区域
│   ├── 文字编辑（标题/副标题/作者）
│   ├── 背景选择
│   │   ├── 纯色渐变
│   │   ├── 内置背景图库
│   │   └── 自定义上传
│   └── 装饰元素
│       ├── Emoji
│       └── 图形
├── 比例切换
└── 底部操作栏：保存 / 复制 / 分享
```

## 5. 优先级建议

### P0（必须）
- 比例切换（4种尺寸）
- 文字编辑增强（对齐方式、多行）
- 纯色/渐变扩展（8→16种配色）

### P1（重要）
- 内置背景图库（8个分类）
- 预设模板（8套模板）
- 装饰元素（emoji + 几何图形）

### P2（优化）
- 自定义上传背景
- 复制到剪贴板
- 多尺寸导出
- 配套文案生成

## 6. 风险与约束

| 风险 | 说明 | 应对 |
|------|------|------|
| 背景图版权 | 内置图库需确认版权 | 使用 CC0/自制/开源图库 |
| 包体积增大 | 背景图资源会增加小程序体积 | CDN 加载，本地仅存配置 |
| Canvas 性能 | 多图+多装饰可能卡顿 | 控制元素数量，按需渲染 |

## 7. 附录：竞品参考

- **Canva**：模板丰富，但功能重
- **图怪兽**：国内工具，模板多，付费墙明显
- **创客贴**：专业设计工具，上手成本高

**差异化方向**：保持小程序轻量优势，做"随手就能用"的快速配图工具，而非全功能设计软件。
