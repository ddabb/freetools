# 数据字典

> 记录 FreeTools 项目中所有关键字段、变量、常量的定义和用途，便于理解和协作。

---

## 一、tools.js 配置字段

每个工具在 `config/tools.js` 中的完整配置：

```javascript
{
  id: 'random-selector',          // [必填] 唯一标识，英文小写
  name: '随机选择器',              // [必填] 显示名称，中文
  icon: '🎲',                      // [必填] Emoji 图标
  color: 'blue',                   // [必填] 颜色主题
  url: '/packages/...',            // [必填] 页面路径
  categories: ['数学工具'],        // [必填] 所属分类数组
  keywords: ['随机', '抽奖'],     // [建议] 搜索关键词
  description: '随机选择器...'     // [建议] 简短描述
}
```

### 颜色主题映射

| color 值 | 实际颜色 | 使用场景 |
|----------|----------|----------|
| blue | #3498db | 默认主色 |
| green | #27ae60 | 成功/正向 |
| orange | #e67e22 | 警告/强调 |
| red | #e74c3c | 危险/错误 |
| purple | #9b59b6 | 特殊/创意 |
| dark | #34495e | 深色工具 |
| light | #95a5a6 | 辅助/次要 |

## 二、Storage Key 规范

微信存储键名前缀约定：

| Key 前缀 | 数据类型 | 过期时间 | 示例 |
|----------|----------|----------|------|
| `cdn_` | CDN 数据（自动清理） | — | `cdn_idiom_index_data`、`cdn_daily_sudoku`、`cdn_24point_questions` |
| `kb_` | 知识库相关 | 7天 | `kb_category_xxx` |
| `search_` | 搜索历史 | 30天 | `search_history` |
| `recentTools` | 最近使用工具 | 永久 | 无前缀 |
| `savedPasswords` | 密码历史 | 永久 | 无前缀 |
| `recentColors` | 颜色历史 | 永久 | 无前缀 |
| `24point_stats` | 24点统计 | 永久 | 无前缀 |
| `useCount` | 使用统计 | 永久 | 无前缀 |

## 三、CDN 路径约定

| 用途 | CDN 路径格式 |
|------|-------------|
| 知识库 | `/know/index.json` |
| 版本日志 | `/data/changelog.json` |
| 数独题目 | `/data/sudoku-presets.json` |
| 24点题目 | `/data/24point-questions.json` |
| 成语索引 | `/data/idiom-solitaire/*.json` |

**完整 URL 格式**：
```
https://cdn.jsdelivr.net/gh/ddabb/FreeToolsPuzzle@main/<路径>
```

## 四、Page data 字段约定

### 4.1 通用状态字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `loading` | boolean | 加载中状态 |
| `loadError` | string | 加载错误信息 |
| `activeTab` | number | 当前 Tab 索引 |

### 4.2 输入输出字段

| 模式 | 输入字段 | 输出字段 |
|------|----------|----------|
| 单输入 | `inputValue` | `result` |
| 多输入 | `param1`, `param2` | `resultList` |
| 列表查询 | `queryInput` | `queryResults` |

### 4.3 对战类字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `battleState` | string | idle/playing/ended |
| `battleTurn` | string | user/ai |
| `battleChain` | array | 接龙链条 |
| `battleTimer` | number | 剩余秒数 |
| `battleResult` | string | win/lose/timeout |

## 五、API 响应格式约定

### 5.1 成功响应

```javascript
{
  success: true,
  data: { ... }
}
```

### 5.2 失败响应

```javascript
{
  success: false,
  error: '错误描述'
}
```

### 5.3 CDN 请求

```javascript
// 成功
res.statusCode === 200 && res.data

// 失败
wx.showToast({ title: '加载失败', icon: 'none' })
```
