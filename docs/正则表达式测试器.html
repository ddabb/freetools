正则表达式提示

  • . - 匹配任意单个字符（除换行符）
  • \d - 匹配数字 (0-9)
  • \w - 匹配单词字符 (字母、数字、下划线)
  • \s - 匹配空白字符 (空格、制表符等)
  • ^ - 匹配字符串开头
  • $ - 匹配字符串结尾
  • [abc] - 匹配 a、b 或 c 中的任意一个
  • a|b - 匹配 a 或 b
  • * - 匹配 0 次或多次
  • + - 匹配 1 次或多次
  • {3} - 匹配恰好 3 次
