# 我用业余时间做了一个开源小程序，69个工具全免费

---

你手机里有多少个工具类 App？

计算器、单位换算、二维码生成、随机密码……每次用都要切来切去，用完就忘在哪了，下次还得再找。

我就是这样的。

与其一直「凑合用」，不如自己做一个——**把所有常用工具塞进一个微信小程序，用完直接关掉，不占内存不占桌面**。

这就是「随身工具宝」诞生的故事。

---

## 69 个工具，能用的有多少？

说实话，不是每个工具我每天都用。

但关键是：**想用的时候，不用专门下载 App，不用注册账号，打开微信就能用**。

按类别分：

**🧮 数学党必备**（11个）
数独求解器、质因数分解、最大公约数最小公倍数、24点、排列组合……考试复习的时候特别实用。

**📝 文字工作者**（11个）
Markdown 预览、文本差异对比、正则测试、字数统计——写东西的时候经常要用。

**🔄 生活刚需**（11个）
长度、重量、温度、面积、体积、速度、压力、能量……单位换算这个事，遇到一次烦一次，有了小程序直接查。

**🌿 日常杂项**（17个）
身份证解读、生肖星座、生命倒计时、成语接龙……有些工具不常用，但用到的时候特别爽。

**💰 金融相关**（3个）
房贷计算器、退休金计算器——算钱的事还是自己来靠谱。

## 怎么做到的

微信小程序有包体积限制，主包 2MB，总包 20MB。69 个工具全塞进去早就超了。

解决方案是**分包加载**——把工具按类型分成 12 个包，用到哪个才加载哪个。

主包只保留首页，800KB 左右，轻松过关。

成语接龙有几万条成语数据，也不可能打包进小程序。做法是**数据放 CDN，按需拉取**，本地缓存一次，后面直接读缓存。

---

## 一些有意思的工具

**数独求解器**
解不出来的数独，输进去一键求解。支持候选数提示，显示每个格子可能的数字。用的是回溯算法，毫秒级出结果。

**成语接龙**
三种模式：查成语释义、人机对战、生成接龙链。数据从 CDN 拉取，不占包体积。

**文字转图片**
做公众号配图、写朋友圈文案，有时候需要把文字转成图片。一键生成，支持自定义字体、颜色、背景。

## 开源的意义

代码扔到 GitHub 上了：[github.com/Free2Know/freetools](https://github.com/Free2Know/freetools)

说实话，不指望靠这个赚钱。就是几个实际好处：

**代码有备份**  
本地硬盘万一出问题，GitHub 就是第二份保险。

**别人能复用**  
有些工具逻辑比较独立，比如数独求解器，直接 copy 过去改改就能用。

**集思广益**  
自己想到的功能有限，社区里有人提个 idea，说不定就加上了。

**简历加分**  
持续维护的开源项目，比培训机构的 demo 项目有说服力多了。

## 参与门槛

想加新工具？三步：

1. 在对应分包下新建页面
2. 在 `config/tools.js` 注册工具信息
3. 在 `app.json` 添加页面路径

代码规范不复杂，微信小程序原生开发，没用 Taro、uni-app 这类框架——这个项目用原生足够。

想提 bug 或者有好想法？开个 Issue 就行。
