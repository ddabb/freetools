# 一个产品经理的深夜：为什么我还在写代码

> 白天做产品需求，晚上写小程序代码。这是热爱，还是逃避？

---

## 一、深夜 11 点，我还在写代码

### 1. 一个普通的晚上

```
22:00 下班回家
22:30 洗完澡，躺床上
23:00 打开电脑，开始写代码
01:00 关电脑，睡觉
```

这是我过去两个月的常态。

### 2. 我在写什么？

FreeTools，一个微信小程序。

- 白天：做产品经理的工作
- 晚上：写小程序的代码

**同事问我：「你白天不是已经够累了吗？为什么晚上还要写代码？」**

### 3. 我的回答

> 「因为我喜欢。」

但这个回答，我自己都不太相信。

---

## 二、我为什么「喜欢」写代码？

### 1. 掌控感

白天做产品经理：

```
提需求 → 等开发评估 → 等排期 → 等开发 → 等测试 → 等上线
```

晚上写代码：

```
有想法 → 写代码 → 立刻看到效果 → 立刻用上
```

**白天是「等待」，晚上是「掌控」。**

### 2. 创造感

白天做产品经理：

```
我提需求 → 别人实现 → 成果是「我们的」
```

晚上写代码：

```
我设计 → 我实现 → 成果是「我的」
```

**白天是「协作」，晚上是「创造」。**

### 3. 即时反馈

白天做产品经理：

```
提需求 → 等两周 → 才能看到效果
```

晚上写代码：

```
写代码 → 保存 → 立刻看到效果
```

**白天是「延迟满足」，晚上是「即时满足」。**

---

## 三、深夜写代码的「逃避」

### 1. 逃避什么？

**逃避白天的无力感：**

- 需求被砍
- 排期被推
- 方案被否
- 产品被改

白天，我的很多想法，无法落地。

晚上，我的所有想法，都能实现。

### 2. 逃避什么？

**逃避人际的消耗：**

白天：
- 和开发撕排期
- 和设计撕方案
- 和运营撕需求
- 和老板撕优先级

晚上：
- 只有我和代码
- 没有撕，只有写

### 3. 逃避什么？

**逃避「不够好」的自己：**

白天：
- 产品做出来，用户不满意
- 需求提出来，开发说做不了
- 方案拿出来，老板说不够好

晚上：
- 代码写出来，立刻能用
- 功能做出来，自己满意
- 产品发出去，有人点赞

---

## 四、深夜写代码的「热爱」

### 1. 热爱什么？

**热爱「创造」的过程：**

```
有一个想法 → 把它变成现实 → 看到有人用 → 感到满足
```

这个过程，让我觉得自己「活着」。

### 2. 热爱什么？

**热爱「解决问题」的感觉：**

```
遇到一个 Bug → 找到原因 → 修复 → 成功
```

这个过程，让我觉得自己「有用」。

### 3. 热爱什么？

**热爱「被认可」的瞬间：**

```
用户说「好用」
用户说「谢谢」
用户说「分享给朋友了」
```

这些瞬间，让我觉得自己「有价值」。

---

## 五、热爱还是逃避？

### 1. 我问了自己三个问题

**问题 1：如果没人用，我还会写吗？**

答案：会。

因为我做给自己用的。

**问题 2：如果很累，我还会写吗？**

答案：会。

因为写代码让我放松。

**问题 3：如果明天要早起，我还会写吗？**

答案：会。

因为深夜是我唯一属于自己的时间。

### 2. 我的结论

**既是热爱，也是逃避。**

热爱：
- 热爱创造
- 热爱解决问题
- 热爱被认可

逃避：
- 逃避白天的无力
- 逃避人际的消耗
- 逃避不够好的自己

**但这两种，都是真实的我。**

---

## 六、深夜写代码教会我的事

### 1. 创造比消费更快乐

消费：
- 刷短视频 → 快乐 5 分钟
- 看剧 → 快乐 1 小时

创造：
- 写代码 → 快乐 3 小时
- 做产品 → 成就感 1 个月

**创造带来的满足感，比消费持久。**

### 2. 掌控感是幸福感的重要来源

白天的无力感：
- 我说了不算
- 我做了没用
- 我想的不被采纳

晚上的掌控感：
- 我说了算
- 我做了有用
- 我想的就是结果

**「我能掌控」，是幸福感的重要来源。**

### 3. 找到「属于自己的时间」

白天：
- 属于公司
- 属于同事
- 属于用户

深夜：
- 属于自己

**每个人都需要「属于自己的时间」。**

## 七、写给同样深夜写代码的你

### 1. 你不是一个人

深夜写代码的人：
- 有的是因为热爱
- 有的是因为逃避
- 有的是因为只有这个时间属于自己

**不管哪种，你都不是一个人。**

### 2. 这不是坏事

深夜写代码：
- 不是「工作狂」
- 不是「逃避现实」
- 是「找到自己的价值」

**只要不影响健康，这不是坏事。**

### 3. 但要注意边界

如果：
- 影响了睡眠
- 影响了白天工作
- 影响了人际关系

**那就需要调整了。**

## 八、写在最后

**深夜 11 点，我还在写代码。**

这是热爱，也是逃避。

但更重要的是：

**这是属于我自己的时间。**

在这段时间里：
- 我可以创造
- 我可以掌控
- 我可以成为「我想成为的人」

**深夜写代码，不是为了逃避白天，而是为了找回自己。**
