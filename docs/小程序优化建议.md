## ⚠️ 待完成优化

### P1 - 高优先级

#### 1. NPM依赖统一 ⏳
**问题**：各分包有独立的 node_modules
**方案**：整合到根目录
**工作量**：2小时

```diff
# 当前结构
freetools/
├── packages/
│   ├── life/node_modules/
│   ├── text/node_modules/
│   └── math/node_modules/
└── root/

# 优化后
freetools/
├── node_modules/          # 统一依赖
└── packages/              # 无独立node_modules
```

#### 6. 错误处理 ⏳
**问题**：错误处理不统一
**方案**：全局错误处理
**工作量**：2小时

```javascript
// app.js 中添加
App({
  onError(err) {
    console.error('全局错误:', err);
    // 上报错误日志
  }
});

// 页面中添加 try-catch
Page({
  onLoad() {
    try {
      // 业务代码
    } catch (err) {
      console.error('页面错误:', err);
    }
  }
});
```
