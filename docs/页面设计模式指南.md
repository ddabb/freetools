# 页面设计模式指南

> 记录 FreeTools 各工具页面的设计模式和可复用方案，供新功能开发参考。

## 一、设计原则

1. **先功能后美观**：先保证可用，再优化体验
2. **复用优于创新**：相同类型的页面采用统一的设计模式
3. **克制配色**：单色系为主，渐变为辅，避免花哨

## 三、输入型页面

适用于：质数判断、奇偶判断、进制转换、单位换算等

### 3.1 结构

```
┌──────────────────────────────────┐
│  [标题]                           │
│  [简短说明]                        │
├──────────────────────────────────┤
│  [输入框1]         ← 实时计算    │
├──────────────────────────────────┤
│  [结果区域]                       │
│  结果1 / 结果2 / 结果3             │
└──────────────────────────────────┘
```

### 3.2 特点

- **无按钮**：输入即计算，移除"计算"按钮
- **实时响应**：`bindinput` 实时触发计算
- **结果分区**：多个结果分开展示，便于阅读

### 3.3 代码模板

```wxml
<view class="container">
  <view class="card">
    <view class="section-title">输入</view>
    <input class="main-input" type="number" bindinput="onInput" placeholder="请输入数字" />
  </view>

  <view class="card result-card">
    <view class="section-title">结果</view>
    <view class="result-row">
      <text class="result-label">判断：</text>
      <text class="result-value">{{result}}</text>
    </view>
  </view>
</view>
```

```wxss
.main-input {
  width: 100%;
  height: 96rpx;
  font-size: 40rpx;
  text-align: center;
  border-bottom: 2rpx solid var(--primary);
}

.result-card {
  background: linear-gradient(135deg, #f0f8ff, #fff);
}
```

## 四、表单型页面

适用于：房贷计算、退休金计算、亲戚计算器等

### 4.1 结构

```
┌──────────────────────────────────┐
│  [标题]                           │
├──────────────────────────────────┤
│  [标签1]        [输入框]          │
│  [标签2]        [输入框]          │
│  [标签3]        [下拉选择]         │
├──────────────────────────────────┤
│  [结果展示区]                      │
│  ┌──────────┐ ┌──────────┐       │
│  │ 月供     │ │ 总利息   │       │
│  │ ¥5,432  │ │ ¥653,214 │       │
│  └──────────┘ └──────────┘       │
└──────────────────────────────────┘
```

### 4.2 特点

- **多字段表单**：每个字段有独立标签
- **计算结果卡片**：重要结果用卡片突出显示
- **实时计算**：参数变化立即刷新结果

### 4.3 代码模板

```wxml
<view class="form-row">
  <text class="form-label">贷款金额（万）</text>
  <input class="form-input" type="digit" bindinput="onLoanChange" value="{{loanAmount}}" />
</view>
```

```wxss
.form-row {
  display: flex;
  align-items: center;
  justify-content: space-between;
  padding: 24rpx 0;
  border-bottom: 1rpx solid #eee;
}

.form-label {
  color: var(--text-secondary);
  font-size: 28rpx;
}

.form-input {
  text-align: right;
  color: var(--text-primary);
  font-size: 32rpx;
}
```

## 五、列表型页面

适用于：成语接龙结果、数独题目列表等

### 5.1 结构

```
┌──────────────────────────────────┐
│  [标题]           [操作按钮]      │
├──────────────────────────────────┤
│  ┌─────────────────────────────┐ │
│  │ 成语A              [尾字]   │ │ ← 可点击继续
│  └─────────────────────────────┘ │
│  ┌─────────────────────────────┐ │
│  │ 成语B              [尾字]   │ │
│  └─────────────────────────────┘ │
└──────────────────────────────────┘
```

### 5.2 特点

- **点击区域分离**：整行显示详情，尾字点击继续接龙
- **高亮尾字**：用彩色标签标识可点击的尾字
- **复制按钮**：便于用户复制结果

### 5.3 代码模板

```wxml
<view class="result-item" bindtap="onItemTap" data-word="{{item}}">
  <text class="result-word">{{item}}</text>
  <text class="result-char" data-next="{{item}}" catchtap="onContinueChain">
    {{item_last_char}}
  </text>
</view>
```

## 六、对战型页面

适用于：成语接龙对战、24点对战等

### 6.1 结构

```
┌──────────────────────────────────┐
│  [系统出题] / [你的回合]          │
│  当前成语：心心相印                │
│  请接："印"字开头的成语           │
├──────────────────────────────────┤
│  ┌──────────────────────────────┐│
│  │ 人机对话气泡                   ││
│  └──────────────────────────────┘│
│  ┌──────────────────────────────┐│
│  │ 用户对话气泡                   ││
│  └──────────────────────────────┘│
├──────────────────────────────────┤
│  [输入框]              [确认]    │
└──────────────────────────────────┘
```

### 6.2 特点

- **回合指示**：明确当前是谁的回合
- **对话气泡**：左右区分系统/用户
- **倒计时**：增加紧张感

## 七、Canvas 图片生成页面

适用于：文案生图、公众号配图、海报生成器等

### 7.1 结构

```
┌──────────────────────────────────┐
│  [编辑区]           [实时预览]    │
│  ┌──────────────┐ ┌────────────┐ │
│  │ 背景色选择   │ │  [Canvas]  │ │
│  │ 文字输入     │ │  [图片]    │ │
│  │ 字体选择     │ │  预览      │ │
│  └──────────────┘ └────────────┘ │
├──────────────────────────────────┤
│  [保存图片]        [复制图片]     │
└──────────────────────────────────┘
```

### 7.2 技术要点

- Canvas 宽度建议 ≤ 750px（适配 Retina 屏）
- 使用 `canvasToTempFilePath` 导出
- 字体需先 `loadFont` 加载
- 导出质量 `quality: 1`

## 八、响应式适配

### 8.1 字号规范

| 场景 | 推荐字号 |
|------|----------|
| 页面标题 | 36-40rpx |
| 卡片标题 | 32rpx |
| 正文内容 | 28rpx |
| 辅助说明 | 24rpx |
| 标签/角标 | 20rpx |

### 8.2 间距规范

```css
/* 容器内边距 */
padding: 0 24rpx;

/* 卡片内边距 */
padding: 24rpx;

/* 元素间距 */
margin-bottom: 20rpx;

/* 行间距 */
line-height: 1.6;
```

### 8.3 圆角规范

```css
/* 按钮 */
border-radius: 8rpx;

/* 卡片 */
border-radius: 16rpx;

/* 输入框 */
border-radius: 8rpx;

/* 标签 */
border-radius: 4rpx;
```
