# 数独求解器技术方案

> 记录数独求解器从算法到 Excel 导出的完整技术实现。

## 一、需求背景

用户需要一个能快速求解数独的工具，支持手动输入题目、自动求解、数独游戏和 Excel 导出。

## 二、算法设计

### 2.1 求解算法：舞蹈链（Dancing Links）+ 回溯

数独本质上是一个精确覆盖问题（Exact Cover Problem），舞蹈链是解决这类问题的高效算法。

```
Dancing Links 核心思想：
1. 用十字链表表示稀疏矩阵
2. 每个节点保存上下左右四个指针
3. 删除/恢复节点只需修改指针，时间复杂度 O(1)
4. 深度优先搜索 + 回溯，极速求解
```

### 2.2 实现细节

```javascript
class DancingLinks {
  // 16个约束条件（4×9=36列，但实际用9×9=81列）
  // 每填一个数字，触发4个约束检查

  cover(col) { /* 移除列及关联行 */ }
  uncover(col) { /* 恢复列及关联行（逆向操作） */ }
  search(k) { /* 递归求解 */ }
}
```

## 三、Excel 导出方案

### 3.1 技术选型

使用 `SheetJS`（xlsx-js-style）实现 Excel 生成。

### 3.2 样式处理

xlsx-js-style 支持单元格背景色、字体加粗等样式，但微信小程序打包器会扫描 `require()` 调用并报错。

**解决方案**：从源码层面 patch 掉所有 `require()` 调用：

```javascript
const patched = xlsxBundle.replace(/require\(.*?\)/g, 'null');
```

Patch 后验证：
- `version`: 0.18.5
- `style_version`: 1.2.0
- 样式正确写入 xlsx 文件

### 3.3 导出格式

```
┌───┬───┬───┬───┬───┬───┬───┬───┬───┐
│ 1 │ 2 │ 3 │   │ 4 │ 5 │ 6 │   │ 7 │
├───┼───┼───┼───┼───┼───┼───┼───┼───┤
│   │ 8 │   │ 9 │   │   │ 1 │   │   │
│  ...                                 │
└───┴───┴───┴───┴───┴───┴───┴───┴───┘
```

- 固定列宽 10
