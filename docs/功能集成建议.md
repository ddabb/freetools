# FreeTools 功能集成建议（本地优先版）

> 创建时间：2026-04-17
> 约束：无后台数据库，所有数据存储在用户本地
> 原则：轻量、离线友好、隐私保护

---

## 一、核心原则

### 1.1 技术约束

| 约束项 | 说明 |
|--------|------|
| 无后台数据库 | 不使用云开发、不自建服务器 |
| 数据本地存储 | 所有用户数据存于 Storage |
| API 按需调用 | 第三方 API（AI/OCR）直接调用，不存储结果 |
| CDN 静态数据 | 知识库、成语等数据通过 CDN 分发 |

### 1.2 可用能力

| 能力 | 来源 | 成本 |
|------|------|------|
| 本地存储 | wx.setStorageSync | 免费，上限 10MB |
| 语音识别 | wx.startRecord + translateVoice | 免费，1万次/月 |
| 语音合成 | 微信同声传译插件 | 免费 |
| 文件导出 | wx.saveFile | 免费 |
| Canvas 绘图 | wx.createCanvasContext | 免费 |
| 第三方 AI API | DeepSeek/通义千问 | 按量计费 |
| OCR API | 百度/腾讯 OCR | 有免费额度 |

**价值：** 夜间使用护眼，跟随系统自动切换

**方案：**
- CSS 变量控制主题色
- Storage 记录用户偏好

**技术方案：**
```css
/* app.wxss */
page {
  --bg-primary: #ffffff;
  --bg-secondary: #f5f5f5;
  --text-primary: #333333;
  --text-secondary: #666666;
  --border-color: #e8e8e8;
}

/* 暗黑模式 */
page.dark {
  --bg-primary: #1a1a1a;
  --bg-secondary: #2d2d2d;
  --text-primary: #e5e5e5;
  --text-secondary: #999999;
  --border-color: #404040;
}
```

```js
// 检测系统主题
wx.getSystemInfo({
  success(res) {
    const isDark = res.theme === 'dark';
    if (isDark) {
      this.setData({ theme: 'dark' });
    }
  }
});

// 监听主题变化
wx.onThemeChange((res) => {
  this.setData({ theme: res.theme });
});
```

## 五、数据管理优化

### 5.1 本地存储规范

**命名规范：**
```js
// 工具相关：tool_{toolId}_{key}
tool_sudoku_history    // 数独历史记录
tool_idiom_favorites   // 成语收藏

// 用户设置：user_{key}
user_theme             // 用户主题偏好
user_favorites         // 用户收藏的工具

// 缓存数据：cdn_{key}
cdn_idiom_data         // 成语数据缓存
cdn_knowledge_articles // 知识库文章缓存
```

**容量管理：**
```js
// 定期清理过期数据
function cleanExpiredData() {
  const info = wx.getStorageInfoSync();
  if (info.currentSize > 7000) { // 超过 7MB
    // 清理最旧的 CDN 缓存
    const cdnKeys = info.keys.filter(k => k.startsWith('cdn_'));
    // 按 timestamp 排序，删除最旧的
  }
}
```

### 5.2 离线体验优化

**策略：**
1. 核心工具数据预加载到本地
2. CDN 数据优先读缓存
3. 网络失败时使用缓存数据 + 静默失败

**实现：**
```js
// 检测网络状态
wx.getNetworkType({
  success(res) {
    if (res.networkType === 'none') {
      // 无网络，显示离线模式
      this.setData({ offlineMode: true });
    }
  }
});

// 加载数据优先使用缓存
async function loadData(url) {
  const cacheKey = `cdn_${url}`;
  const cached = wx.getStorageSync(cacheKey);
  
  if (cached) {
    // 立即返回缓存
    return cached;
  }
  
  // 后台请求更新
  wx.request({
    url,
    success(res) {
      wx.setStorageSync(cacheKey, res.data);
    }
  });
}
```

## 九、风险与注意事项

| 风险 | 影响 | 缓解方案 |
|------|------|----------|
| AI API 成本超预算 | 功能下线 | 设置每日调用上限 |

| 第三方 API 不稳定 | 功能不可用 | 多服务商备份、优雅降级 |
