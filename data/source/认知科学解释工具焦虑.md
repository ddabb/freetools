> 从认知科学角度，解释为什么我们总是「收藏了不用」。

---

## 一、一个普遍的现象

### 1. 你的收藏夹

打开你的微信收藏：

- 收藏的文章：50 篇？100 篇？
- 看过的文章：10 篇？5 篇？

### 2. 你的 App 列表

打开你的手机：

- 安装的 App：50 个？100 个？
- 每天打开的 App：10 个？5 个？

### 3. 你的工具箱

我做的 FreeTools 有 57 个工具：

- 用户访问过的：平均 3.2 个
- 经常使用的：平均 1.8 个

**「收藏了不用」，是一个普遍现象。**

---

## 二、大脑的「记忆保护机制」

### 1. 米勒定律：7±2

1956 年，心理学家乔治·米勒发现：

> **人的工作记忆容量是 7±2 个单位。**

这意味着：
- 一次只能记住 5-9 个东西
- 超过的，要么忘掉，要么存入长期记忆

**你的大脑只能同时处理 5-9 个信息。**

### 2. 为什么是 7±2？

这是大脑的保护机制：

```
如果大脑能记住所有信息 → 信息过载 → 瘫痪
如果大脑只能记住关键信息 → 保持清晰 → 正常运作
```

**「记不住」不是缺点，是大脑的自我保护。**

### 3. 希克定律：选择越多，反应越慢

1952 年，心理学家威廉·希克发现：

> **选项越多，决策时间越长。**

公式：`决策时间 = log2(选项数量) × 常数`

如果你有 57 个工具：
- 用户要扫描 57 个选项
- 大脑要判断「哪个是我要的」
- 判断时间 × 57 ≈ 认知疲劳

**选择太多 = 认知负担 = 放弃选择。**

## 三、「工具焦虑」的真相

### 1. 为什么我们喜欢「收集」？

大脑的多巴胺机制：

```
看到有用的信息 → 多巴胺分泌 → 感觉「以后用得上」 → 收藏
↓
收藏后 → 多巴胺消失 → 没有再看 → 遗忘
```

**「收藏」这个动作本身，已经给你带来了满足感。**

### 2. 为什么「收藏」后不用？

大脑的能量守恒原则：

```
收藏 → 花费能量 0.1
以后再看 → 花费能量 1.0
大脑选择 → 能量最低的路径 → 不看
```

**大脑会自动选择「最省力」的路径。**

### 3. 为什么「工具多」反而焦虑？

选择悖论：

```
有 5 个工具 → 容易选择 → 用 1 个 → 满足
有 57 个工具 → 难以选择 → 犹豫 → 不用 → 焦虑
```

**选择越多，决策越难，焦虑越强。**

## 四、如何对抗「工具焦虑」？

### 1. 接受「记不住」的事实

**真相：**
- 你记不住所有工具
- 你也不需要记住所有工具
- 记住 5 个核心工具就够了

**接受：**
- 不是你记性不好
- 是你的大脑在保护你

### 2. 建立「工具使用原则」

**原则 1：一个需求只用一个工具**

```
需要计算日期 → 只用「日期计算器」
需要生成二维码 → 只用「二维码生成器」
```

**原则 2：搜索优于分类**

```
不要记住工具在哪
只需要搜索你想做什么
```

**原则 3：不收藏，直接用**

```
看到有用的工具 → 打开用 → 用完即走
不要「收藏以后用」→ 因为你不会看
```

### 3. 减少选择，增加行动

**方法 1：删掉不用的 App**

```
打开你的手机
删除最近 30 天没打开的 App
你会发现：你不需要那么多
```

**方法 2：精简收藏夹**

```
打开你的收藏
删除超过 3 个月没看的
你会发现：你根本不会看
```

**方法 3：一个工具用到底**

```
找到一个顺手的工具
用它 3 个月
形成习惯
不要换
```

## 五、产品设计者的视角

### 1. 我做 FreeTools 时的错误

**错误 1：以为工具越多越好**

- 实际上：用户只需要 5 个
- 57 个工具 = 57 个选择 = 决策困难

**错误 2：以为用户会探索**

- 实际上：用户只会搜索
- 用户不会主动发现新工具

**错误 3：以为「收藏」等于「使用」**

- 实际上：收藏 ≠ 使用
- 收藏只是「缓解焦虑」，不是「解决问题」

### 2. 我学到的教训

**教训 1：少即是多**

- 做好 5 个核心工具 > 做烂 50 个工具
- 用户宁愿「没有选择」，也不愿意「选择困难」

**教训 2：搜索是核心入口**

- 不要指望用户「记住」你的功能
- 让用户「搜索」到功能

**教训 3：用完即走**

- 不要绑定用户
- 让用户用完就关
- 不要强迫用户「收藏」

## 六、写给用户的建议

### 1. 你的大脑很聪明

**它知道哪些信息重要，哪些信息不重要。**

**「记不住」不是你的错，是大脑在帮你筛选。**

### 2. 你的时间是有限的

**你不可能学会所有工具。**

**你只需要掌握对你真正有用的那 5 个。**

### 3. 你的选择是有效的

**不选择也是一种选择。**

**放弃 50 个工具，专注 5 个工具，这才是最聪明的选择。**

---

## 七、写在最后

**工具焦虑的本质，是「选择太多」。**

**解决方法是「减少选择」。**

- 米勒定律：大脑只能记住 7±2 个
- 希克定律：选择越多，决策越慢
- 能量守恒：大脑会选择最省力的路径

**所以，不要怪自己「记不住」，要感谢大脑在保护你。**
