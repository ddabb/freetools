## 核心功能

### 财务工具
- **房贷计算** - 计算房贷月供、利息和还款总额
- **价格对比** - 对比多个商品的单价和性价比
- **退休金计算** - 计算退休所需的储蓄金额

### 生活工具
- **人生A4纸** - 计算已活天数和剩余预期寿命
- **电池健康** - 查看设备电池健康状态
- **万年历** - 查看日历和农历日期
- **时间工具** - 秒表、计时器和时区转换
- **时区转换** - 在不同时区之间转换时间
- **日期差计算** - 计算两个日期之间的天数差
- **旅行工具** - 旅行规划和辅助工具

### 实用工具
- **生成二维码** - 生成文本和链接的二维码
- **单位换算** - 各种单位的快速换算
- **密码生成** - 生成安全的随机密码
- **身份证验证** - 验证身份证号码的有效性
- **文本工具** - 文本处理和转换工具
- **Markdown转换** - Markdown文档转换工具
- **奇偶判断** - 判断数字的奇偶性
- **闰年判断** - 判断是否为闰年
- **中文检测** - 检测文本中是否包含中文

### 健康工具
- **健康计算** - 计算BMI和健康指标

### 数学工具
- **公约数·公倍数** - 计算两个数的最大公约数和最小公倍数

- **进制转换** - 在不同基数间转换数字，支持二进制、十六进制、Base64等

- **YAML·JSON转换器** - 在YAML和JSON格式之间进行相互转换
- **正则校验** - 在线测试和调试正则表达式，支持高亮显示匹配结果

### 文本工具

- **Markdown预览器** - 实时预览MD文档效果
- **文本对比** - 高亮显示文本差异

### 数据工具
- **数据统计器** - 数值统计与分布分析

### 图表工具
- **图表生成器** - 多种图表一键生成

### 开发工具
- **时间戳转换器** - Unix时间戳互转
- **颜色转换器** - RGB/Hex/HSL色彩转换

### 安全工具
- **密码生成** - 生成安全的随机密码
- **身份证验证** - 验证身份证号码的有效性

- **搜索历史** - 记录搜索历史，方便快速访问

## 开发指南

### 添加新工具

1. 在 `packages/` 对应分包目录下创建工具页面
2. 在 `config/tools.js` 中添加工具配置
3. 如需要，在 `utils/` 中添加工具函数

### 开发规范

- 使用 CommonJS 模块化（require/module.exports）
- 统一代码风格：ES6 方法简写、let/const
- 所有输入必须验证和错误处理
- 优先使用已有的工具函数库

### 分包策略

- **主包**：核心页面（首页、发现、关于）
- **分包**：按功能分类，减少首次加载时间
  - `chart/` - 图表工具分包
  - `converter/` - 转换工具分包
  - `data/` - 数据工具分包
  - `design/` - 设计工具分包
  - `developer/` - 开发工具分包
  - `financial/` - 财务工具分包
  - `food/` - 饮食工具分包
  - `health/` - 健康工具分包
  - `life/` - 生活工具分包
  - `math/` - 数学工具分包
  - `text/` - 文本工具分包
  - `travel/` - 旅行工具分包
  - `unit/` - 单位换算工具分包
  - `utility/` - 实用工具分包

## 注意事项

1. 不要使用 ES6 import/export 语法，使用 require/module.exports
2. 避免使用展开运算符 `...`，使用 Object.assign()
3. 新增工具必须更新 `config/tools.js`
4. 所有工具函数优先使用 `utils/` 中的现有工具
5. 必须添加输入验证和错误处理
