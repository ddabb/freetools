## 🎯 核心特性

- **题库 CDN 架构**：游戏题库走独立 CDN 分发，支持大文件（15MB+索引）
- **分包加载策略**：主包1.5MB，全量3.3MB，首屏加载提速60%
- **老年友好模式**：大字号、高对比度、操作简化
- **本地缓存**：关键数据缓存，减少重复请求
- **AI 辅助开发**：采用 VibeCoding 模式，开发效率提升3倍

## 🚀 开发指南

### 技术栈
- 微信小程序原生框架
- JavaScript (ES5/ES6)，WXML/WXSS
- 模块化开发（require/module.exports）
- Canvas 2D 游戏渲染

### 添加新工具
1. 在 `packages/` 对应分包下创建页面
2. 更新 `app.json` 注册页面
3. 更新 `config/tools-full.js` 添加工具入口
4. 提交 PR / 推送 GitHub

### 分包策略
- **主包**：核心页面（首页、知识库、关于）
- **工具分包**：按功能分类，首屏加载优化
- **游戏分包**：大题库游戏独立分包，题库走 CDN

## ⚠️ 开发规范

1. 不使用 ES6 import/export（用 require/module.exports）
2. 避免展开运算符 `...`（用 Object.assign()）
3. 题库资源走 CDN，不打包进小程序
4. 新增页面必须注册到 app.json
5. 游戏题库需独立验证（生成器+求解器双重验证）
